# Complete Example Combining all features in one slide. ## The Derivative In calculus, the derivative measures the rate of change of a function. For $f(x) = x^2$, the derivative is: $$f'(x) = \lim_{h \to 0} \frac{(x+h)^2 - x^2}{h} = 2x$$ ## Python Code ```python def derivative(f, x, h=0.0001): """Calculate derivative using finite difference""" return (f(x + h) - f(x)) / h # Test with f(x) = x^2 at x = 3 result = derivative(lambda x: x**2, 3) print(f"f'(3) = {result:.2f}") # Output: ~6.00 ``` The result is approximately $6$, which is the correct derivative of $x^2$ at $x = 3$.
